Module: clipboard-plugin

The Clipboard plugin provides you an interface to set up a copy to OS clipboard using a Flash movie bridge. The ClipBoard class is a node plugin, so any YUI3 node can be given the ability to set the content to the clipboard triggered off a by an event.
